# Medical-data-analysis
对儿童医疗数据进行分析，辅助医生诊断。

代码运行：
--------------
		1、运行medical_predict.py进行预测。
		2、输入相应的age、cure_before（治疗前的疗效，分为9各等级），输入当前诊断。
	
输出结果：
--------------
		1、给出以往最近的5个相似病人用药情况及疗效供医生参考。
		2、模型推荐的药品（共分为西药、中成药、中草药、检查及检验）
		3、输出可能伴随的疾病，起到预防作用（主要利用关联分析找到疾病之间的关系）
